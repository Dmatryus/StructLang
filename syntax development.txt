# Типы блоков
* Объект: septagon: {{}}
* Контейнер: box3d: [[]]
* Свойство: box: []
* Следствие: parallelogram: [{}]

# Типы связей
* Включение: inv: <-
* Объяснение: normal: ->
* Структура: none: --

**Пример строчной записи**:

{{Разделение выборки}} -- {
    [[Выборки]] <- {
        {{Обучающая выборка}} -> [На ней обучается модель]
        {{Тестовая выборка}} -> [На ней происходит оценка модели]
        {{Валидационная выборка}} -> [На ней тестируется окончательно]
    }
    [{Зачем выделять валидационную выборку?}] -> {{Утечка информации}} -> [Процесс конструирования модели связанг с настройкой гиперпараметров] -> [{Настройка гиперпараметров - часть обучения}] -> [{Может случится переобучение в пространстве гиперпараметров, если много раз настраивать модель на тестовой выборке}] -> [{С каждой новой итерацией в модель просачиваентся немного информации о тестовой выборке}]
}
